# 分享遊戲 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** 遊戲頁 `#saveui` 在留言右邊加「分享」鈕，用 `navigator.share`（手機系統分享單），桌機退回複製連結。

**Architecture:** 全部在 `pwa.py` 的 `_SAVE_UI` 內嵌 classic JS（按鈕＋handler）；無新檔、無後端、無 Firestore 規則。

**Tech Stack:** 純前端原生 API（`navigator.share`/`navigator.clipboard`）、Python build（pwa.py）、pytest。

## Global Constraints

- 設計依據：`docs/superpowers/specs/2026-06-22-share-design.md`。
- `#saveui` 順序（左→右）：`存檔(save-open)`、`❤(fav-btn)`、`已遊玩(pt-label)`、`攻略(wt-open)`、`留言(cm-open)`、`分享(share-btn)`——分享最右（留言右邊）。
- 分享內容＝`(window.__WT && window.__WT.title) || document.title` 與 `location.href`。
- 取消/失敗安靜略過；無 clipboard 退回 `prompt`。
- 不動 `web/firestore.rules`、不需新檔、不需 module。
- `web/firebase-config.js`（真實金鑰）維持 gitignored；只動 `pwa.py`、測試。

```python
def test_write_game_pages_injects_share(tmp_path):
```

- [ ] **Step 3: 在 `#saveui` 按鈕列加「分享」鈕（留言右邊）**

在 `pwa.py` 找到（約第 202 行）：

```html
<button id="cm-open">留言</button>
```

改成（在留言之後加分享）：

```html
<button id="cm-open">留言</button><button id="share-btn">分享</button>
```

（注意：`#saveui` 那一行很長且 `cm-open` 只出現一次；只在該按鈕列尾端加，勿動其他。）

- [ ] **Step 4: 在 IIFE 加分享 handler**

在 `pwa.py` 找到（約第 270 行）：

```javascript
panel.querySelector(".sp-close").onclick=spClose;panel.querySelector(".sp-bd").onclick=spClose;
```

在它後面加一段：

```javascript
document.getElementById("share-btn").onclick=function(){
var t=(window.__WT&&window.__WT.title)||document.title;var u=location.href;
if(navigator.share){navigator.share({title:t,url:u}).catch(function(){});}
else if(navigator.clipboard){navigator.clipboard.writeText(u).then(function(){alert("已複製連結");},function(){prompt("複製這個連結：",u);});}
else{prompt("複製這個連結：",u);}};
```

Expected: 無輸出（內嵌 JS 語法正確）。
