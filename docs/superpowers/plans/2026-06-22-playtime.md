# 遊戲頁「玩了多久」（遊玩紀錄 / playtime）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** 在遊戲頁顯示該遊戲累計遊玩時間（「已遊玩 Xh Ym」），時間在 localStorage 本機累計、**只在離開頁面時寫一次 Firestore**。

**Architecture:** 新增 `web/playtime.js`（只在遊戲頁，`import account.js`）；`pwa.py` 注入 `#pt-label` 標籤 + 模組；規則擴充 `web/firestore.rules`。資料存 `users/<uid>/history/<slug>`（私有），用 `increment` 原子累加。

**Tech Stack:** Firebase 9.23.0（gstatic CDN 模組）、純前端 ES module、localStorage、Python build（pwa.py）、pytest。

## Global Constraints

- 設計依據：已核准計畫 `C:\Users\david\.claude\plans\giggly-beaming-finch.md`。
- Firebase SDK 固定 `https://www.gstatic.com/firebasejs/9.23.0/firebase-firestore.js`。
- 私有：`users/<uid>/history/<slug>` 只有 `request.auth.uid == uid` 能讀寫；欄位只允許 `['totalSeconds','lastPlayedAt']`；delete 拆開只看本人（delete 時 request.resource 為 null）。
- **寫入策略**：遊玩過程完全不寫雲端、無門檻、無週期；只在 `visibilitychange→hidden` 或 `pagehide`、且本機緩衝 > 0 時寫一次 `setDoc(..., {merge:true})` 以 `increment(buffer)` 累加；成功才把緩衝扣掉（失敗則留待下次補寫，不遺失/不重複）。
- 計時只在 `document.visibilityState === 'visible'` 時累加；顯示 = 雲端載入值（讀一次）＋ localStorage 緩衝 ＋ 本次 live。
- 未登入：只本機顯示、不讀寫雲端；登入後讀雲端一次、緩衝於下次離開時補寫。
- `web/firebase-config.js`（真實金鑰）維持 gitignored、永不 commit；只動 `firestore.rules`、`playtime.js`、`pwa.py`、測試。

### Task 2: 前端模組 `web/playtime.js`

**Files:** Create `web/playtime.js`

**Interfaces:**
- Consumes: `./account.js`（`db, isReady, currentUser, onAuthChange`）；Firebase 模組 `doc, getDoc, setDoc, serverTimestamp, increment`；DOM `#pt-label`；`window.__WT.slug`。

- [ ] **Step 1: 建立 `web/playtime.js`（完整內容）**

```javascript
// 遊玩時間：在遊戲頁累計「分頁可見」的秒數，離開頁面才寫一次 Firestore（增量）。私有，依賴 account.js。
import {
  doc, getDoc, setDoc, serverTimestamp, increment,
} from 'https://www.gstatic.com/firebasejs/9.23.0/firebase-firestore.js';
import { db, isReady, currentUser, onAuthChange } from './account.js';

const SLUG = (window.__WT && window.__WT.slug) || '';
const label = document.getElementById('pt-label');
const BUF_KEY = 'ep-pt-' + SLUG;

let cloudSeconds = 0; // 雲端已存的總秒數（登入後讀一次）
let live = 0;         // 本次開頁、尚未併入 localStorage 緩衝的記憶體秒數
let flushing = false;

function buffer() { return parseInt(localStorage.getItem(BUF_KEY) || '0', 10) || 0; }
function setBuffer(n) {
  if (n > 0) localStorage.setItem(BUF_KEY, String(n));
  else localStorage.removeItem(BUF_KEY);
}
function fmt(s) {
  s = Math.floor(s);
  const h = Math.floor(s / 3600);
  const m = Math.floor((s % 3600) / 60);
  if (h) return '已遊玩 ' + h + 'h' + (m ? ' ' + m + 'm' : '');
  if (m) return '已遊玩 ' + m + 'm';
  return '已遊玩 ' + s + 's';
}
function render() {
  if (label) label.textContent = fmt(cloudSeconds + buffer() + live);
}

if (SLUG) start();

function start() {
  render();
  // 每秒累加（僅可見時）；每累積 5 秒併入 localStorage 緩衝（本機持久，不碰 DB）
  setInterval(() => {
    if (document.visibilityState === 'visible') { live += 1; render(); }
    if (live >= 5) { setBuffer(buffer() + live); live = 0; }
  }, 1000);

  // 登入後讀一次雲端總時數；登出歸零
  if (isReady()) {
    onAuthChange((u) => {
      if (u) loadCloud(u.uid);
      else { cloudSeconds = 0; render(); }
    });
  }

  // 離開頁面：併入緩衝後寫一次
  document.addEventListener('visibilitychange', () => {
    if (document.visibilityState === 'hidden') flush();
  });
  window.addEventListener('pagehide', flush);
}

async function loadCloud(uid) {
  try {
    const snap = await getDoc(doc(db, 'users', uid, 'history', SLUG));
    cloudSeconds = (snap.exists() && typeof snap.data().totalSeconds === 'number')
      ? snap.data().totalSeconds : 0;
  } catch (e) { cloudSeconds = 0; }
  render();
}

async function flush() {
  if (live > 0) { setBuffer(buffer() + live); live = 0; }
  const u = isReady() ? currentUser() : null;
  const amount = buffer();
  if (!u || amount <= 0 || flushing) return;
  flushing = true;
  try {
    await setDoc(
      doc(db, 'users', u.uid, 'history', SLUG),
      { totalSeconds: increment(amount), lastPlayedAt: serverTimestamp() },
      { merge: true },
    );
    cloudSeconds += amount;
    setBuffer(buffer() - amount); // 只扣已寫量，期間新增的留著
  } catch (e) { /* 失敗：緩衝留著，下次離開再補寫 */ }
  flushing = false;
}
```
